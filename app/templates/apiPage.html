Примеры запросов к API

На этой странице представлены примеры запросов к API - repair-31.ru

GET Запрос

Получение списка всех заявок


Эмулированный пример GET запроса:

fetch('https://repair-31.ru/get-all-applications', {
    method: 'GET',
    headers: {
        'X-API-KEY': 'abc123',
        'X-API-SECRET': 'abc321cba123'
    },
})
    .then(response => response.json())
    .then(data => console.log(data));

POST Запрос

Получение списка всех заявок с фильтром статуса


Эмулированный пример POST запроса:

fetch('https://repair-31.ru/get-all-filter-applications', {
    method: 'POST',
    headers: {
        'X-API-KEY': 'abc123',
        'X-API-SECRET': 'abc321cba123'
    },
    body: JSON.stringify({
        status: 'active'
    })
})
    .then(response => response.json())
    .then(data => console.log(data));
